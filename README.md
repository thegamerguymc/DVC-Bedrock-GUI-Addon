# DVC-Bedrock-GUI-Addon
This project adds a GUI for bedrock that will allow users to use the Simple Voice Chat Discord Bridge with a GUI on Bedrock Edition via an inventory item that opens a menu.


⚠️ THIS PLUGIN/MOD IS CODED BY AI AND IS STILL IN ALPHA/BETA, THERE ARE STILL SOME BUGS.
This plugin makes Simple Voice Chat Discord Bridge much easier to use for Bedrock Edition users (especially console users) by adding an easy to use GUI/Menu that allows for Bedrock Edition users who are on a server with Geyser, Floodgate, and Simple Voicechat Discord Bridge (SVD) be able to use commands from the SVD plugin with a simple user-friendly menu.

Both Simple Voice Chat and Simple Voice Chat Discord Bridge and geyser + floodgate are needed for this plugin to work.

How to use
Bedrock players can simply open the menu easily with an inventory item that they will spawn with (a nether star) to open the menu. Then a menu will pop up showing buttons for all the Simple Voice Chat Discord Bridge (dvc) commands.

Features
An easy to use menu for bedrock users to use Simple Voice Chat Discord Bridge easier (especially for console users)

Allows for Java players to use /voicechat from the Simple Voice Chat plugin on Bedrock users to easily invite them to a voicechat group. (Only works Java to Bedrock currently)

Commands
/dvcui -- Opens the GUI to control Simple Voice Chat Discord Bridge (only works on bedrock)
/bvc -- Opens menu only if config is set to 'COMMAND' or 'BOTH'
